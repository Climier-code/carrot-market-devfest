## Usage

1. Git Clone

```cmd
$ git clone https://github.com/Climier-code/carrot-market-devfest.git
```

2. Install package

```cmd
$ yarn

# or

$ npm install
```

3. Start

```cmd
$ yarn start

# or

$ npm run start
```

4. Deploy

```cmd
$ yarn deploy

# or

$ npm run deploy
```
